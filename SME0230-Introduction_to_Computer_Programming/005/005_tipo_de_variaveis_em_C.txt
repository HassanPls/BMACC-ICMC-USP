Na linguagem C, os dados inteiros podem ser definidos usando short int,
int, long int, long long int, unsigned short int, unsigned int, unsigned long
int ou unsigned long long int.

Na linguagem C, os dados reais podem ser definidos usando float, double
ou long double.

Na linguagem C, os caracteres podem ser definidos usando char ou
unsigned char.

Na linguagem C, as cadeias de caracteres podem ser definidos usando
char[] ou unsigned char[].
